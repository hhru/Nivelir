  • presenting

    The container that presented this container.

    When you present a container modally (either explicitly or implicitly) using the present(_:animated:completion:) method, the container that was presented has this property set to the container that presented it. If the container was not presented modally, but one of its ancestors was, this property contains the container that presented the ancestor. If neither the current container or any of its ancestors were presented modally, the value in this property is nil.

    See also
