  • AnyScreenAction

    An action that performs type erasure by wrapping another action.

    AnyScreenAction is a concrete implementation of ScreenAction that has no significant properties of its own, and passes through elements from its wrapped action.

    Use AnyScreenAction to wrap an actions whose type has details you donâ€™t want to expose across API boundaries, such as different modules. When you use type erasure this way, you can change the underlying acrtion implementation over time without affecting existing clients.

    You can use eraseToAnyAction() and eraseToAnyVoidAction()methods to wrap an action with AnyScreenAction.
